## BYML Editor

Quickly done (probably buggy) byml (v2 for botw) encoder/decoder

#### Decode to JSON
```
java -jar byml-editor-1.0-SNAPSHOT.jar d file.byml file.json
```

#### Encode to BYML
```
 java -jar byml-editor-1.0-SNAPSHOT.jar c file.json file.byml
```
